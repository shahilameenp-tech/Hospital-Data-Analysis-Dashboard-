# Hospital-Data-Analysis-Dashboard-
## Project Description

The Hospital Data Analysis Dashboard is an interactive data visualization project designed to analyze hospital operations, patient trends, and financial performance.

It helps healthcare professionals and administrators make data-driven decisions by providing clear insights into patient admissions, treatment outcomes, and resource utilization.
Project Objectives
•	Monitor hospital performance through key metrics
•	Analyze patient demographics and disease patterns
•	Track revenue and billing trends
•	Improve operational efficiency and patient care
•	Identify areas for improvement in hospital services


 ## Dataset Overview

The dataset contains hospital-related information such as:
•	Patient ID
•	Gender & Age
•	Admission & Discharge Dates
•	Department
•	Disease / Diagnosis
•	Billing Amount
•	Patient Outcome

 #Key Performance Indicators (KPIs)
 
Total number of patients admitted to the hospital.
Tracks number of admissions over time.
Total income generated from patient billing.
Percentage of patients who recovered successfully.


Percentage of patients who did not survive.
Number of patients per department
Revenue generated per department

## Dashboard Features
•	Interactive filters ( Department, Gender,Outcome)
•	Admission trends over time
•	Revenue analysis
•	Patient distribution by disease
•	Department performance comparison
## Dashboard Design
•	Created visualizations:
o	Line Chart → Admission trends

## Key Insights

•	Revenue is concentrated in specific departments
•	Recovery rates vary across diseases and treatments

## Conclusion

The Hospital Data Analysis Dashboard provides a clear and interactive way to monitor hospital performance.

It helps:
•	Improve patient care
•	Optimize hospital resources
•	Support strategic decision-making

Overall, the dashboard transforms raw hospital data into meaningful insights that can enhance efficiency and outcomes
